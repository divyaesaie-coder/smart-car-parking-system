# Smart Car Parking Management System

## Overview
This project is a Python-based Smart Car Parking Management System designed to manage vehicle parking operations efficiently.

## Features

- Vehicle entry system
- Vehicle exit system
- Tracks available parking slots
- Prevents duplicate entries
- Stores car numbers
- Records entry and exit times
- Displays remaining parking slots

## Technologies Used

- Python

## Concepts Used

- Lists
- Loops
- Conditional Statements
- User Input Handling

## How It Works

1. User selects car entry
2. Car number is recorded
3. Parking slot count decreases
4. User exits car
5. Slot count increases
6. System displays remaining slots
